## Prerequisites

To get the most out of this module, you should have:

- Basic programming knowledge (Python or C++)
- Familiarity with Linux command line
- Understanding of basic robotics concepts (helpful but not required)

## Recommended Setup

- **OS**: Ubuntu 22.04 LTS (recommended) or Ubuntu 24.04 LTS
- **ROS 2 Distribution**: Humble Hawksbill (LTS) or Jazzy Jalisco
- **Hardware**: 4GB+ RAM, 20GB+ free disk space
- **Optional**: Virtual machine (VirtualBox/VMware) or Docker
